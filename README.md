# Online Examination System

## Overview

The Online Examination System is a Java-based console application that simulates an online exam environment. The system allows users to log in, update their profile information, take a multiple-choice examination, and view their final score after submission.

## Features

* User Login System
* Profile Update Option
* Multiple Choice Questions (MCQ)
* Automatic Exam Submission
* Time-Limited Examination
* Score Calculation
* Simple Console-Based Interface

## Technologies Used

* Java
* Scanner Class
* Arrays
* Methods
* Conditional Statements
* Loops

## Modules

### 1. Login Module

* Allows users to enter a username and password.
* Grants access to the examination system.

### 2. Update Profile Module

* Enables users to change their username.
* Allows users to update their password.

### 3. Examination Module

* Displays multiple-choice questions.
* Accepts answers from the user.
* Checks answers with the correct options.
* Calculates the final score.

### 4. Timer Module

* Exam duration is limited to 60 seconds.
* Automatically submits the exam when the time limit is reached.

### 5. Logout Module

* Safely exits the examination system.

## How the System Works

1. User enters login credentials.
2. Main menu is displayed.
3. User can:

   * Update Profile
   * Start Exam
   * Logout
4. During the exam, questions are displayed one by one.
5. Answers are evaluated instantly.
6. After completion, the system displays the final score.
7. User can logout from the system.

## How to Run

Compile the program:

```bash
javac task4.java
```

Run the program:

```bash
java task4
```

## Output

* Login Successful
* Profile Update Option
* Online Examination
* Automatic Submission on Time Expiry
* Final Score Display
